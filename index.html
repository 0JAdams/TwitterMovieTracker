    Track Twitter Sentiment For Upcoming Movies

    Project maintained by 0JAdams Hosted on GitHub Pages — Theme by mattgraham

    Intro

    TwitterMovieWatcher is a set of Python scripts for monitoring a stream of tweets in relation to chosen movies. Those tweets are then classified by their overall sentiment. Release date and box office data are pulled from boxofficemojo.com (thanks goes to them for that). The data is compared to find correlations.

    How It Works

    After everything has been initialized and trained, using the tweepy library for Python a twitter stream is initialized. Based on a list of movie keywords, all relevant tweets are pulled live from twitter. Because a Twitter stream returns all tweets related to all of your keywords, those tweets are then checked against a dictionary of keywords to match the keyword in the tweet back to the correct movie.

    As each tweet comes in it is pre-processed. First, each word is stemmed using the Natural Language Tool-Kit (NLTK) for python. This makes the words match our training set (which was also stemmed). The stemming helps us have a more concise feature set for comparing against and results in more accurate sentiment analysis. Using the NLTK library and the scikit-learn library, the tweet is then passed to a series of classification algorithms, which each produce an analysis of the sentiment and vote on whether the sentiment was positive or negative. The tweet ID, time stamp, related movie, and sentiment classification are stored in an SQL database for later reference.

    A simple web scraper that uses the urllib3 and BeautifulSoup4 libraries pulls the release date and daily box office numbers from boxofficemojo.com and stores them in an SQL table.

    All of the sentiment and box office data is then graphed over time to view correlations.

    Current State

    Currently the project is a collection of Python scripts that are mostly separate with each performing one function of process. The code is in place to train the classification algorithms, stream and analyze relevant tweets, store them in an SQL database, and scrape each movie's box office data.
